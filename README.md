# Smart Parking Management System 🚗

A full stack smart parking application built using React.js and Node.js.

---

## Features

- User Login & Register
- Parking Slot Visualization
- Slot Booking
- Payment UI
- QR Code Generation
- Booking History
- Real-Time Slot Updates
- Frontend & Backend Integration

---

## Technologies Used

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js

## How to Run Frontend

```bash
npm start
```

---

## How to Run Backend

```bash
cd backend
npx nodemon server.js
```
